FILES INCLUDED IN THIS DATA DISTRIBUTION PACKAGE:
================================================

 - Comma-delimited data files:
 	- qs_a104_Location_metadata - Sampling location (transect) metadata export
	- qs_a114_Event_metadata - Sampling event metadata export, one event record per transect-year
	- qs_a124_Event_observers - 0-multiple observers associated with each sampling event
	- qs_a134_Phenology_obs - 0-multiple phenology species observations per sampling event
	- qs_a144_Transect_obs - 0-multiple transect segment observations per sampling event

 - Lookup tables (Excel format):
	- tlu_Cover_Class.xlsx - Domain lookup for transect segment vegetation cover classes
	- tlu_Origin_Code.xlsx - Domain lookup for origin categories
	- tlu_Substrate.xlsx - Domain lookup for substrate categories
	- tlu_Veg_Type.xlsx - Domain lookup for transect segment vegetation type categories

 - Documentation:
	- SOP_6_Monitoring_Transects_20140918.docs - Current protocol SOP for transect monitroing, updated most recently in 2014
	- 2014_Prairie_transect_data_form.docx - Current field form, updated most recently in 2014
	- NCCN_Prairie_Veg_Database_Documentation_20160711.docx - Definitions of relational database tables and fields
	- NCCN_Prairie_Veg_ERD_20160711.png - Image showing the entity-relational diagram (ERD) of tables and relationships in the database
	- 2007-2015_Certification_Results_20160711.pdf - Data certification output, including quality assurance queries and results
	- 2015_Data_Certification_Form_VCa03_DRAFT_20160711.docx - Draft data certification summary for 2007-2015
